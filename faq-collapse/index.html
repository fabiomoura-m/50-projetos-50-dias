Perguntas Frequentes

Qual a função do HTML?

Define o significado e a estrutura do conteúdo da web

Para que serve o CSS?

É usado para descrever a apresentação de um documento escrito em HTML ou em XML (incluindo várias linguagens em XML como SVG, MathML ou XHTML).

O que é linguagem de programação?

É um conjunto de comandos, instruções e regras utilizadas para criar um programa de computador.

Qual é o papel da lógica na programação?

Conduzir o raciocínio para desenvolver as soluções.

HTML e CSS são linguagens de programação?

Não. HTML é uma linguagem de marcação e CSS linguagem de estilo.
